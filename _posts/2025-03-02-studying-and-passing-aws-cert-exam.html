Exam Overview

The AWS Certified Cloud Practitioner Exam validates foundational, high-level understanding of AWS Cloud, services, and terminology. It's an excellent starting point on the AWS Certification journey for:

  • Individuals with no prior IT or cloud experience switching to a cloud career
  • IT professionals looking for foundational cloud literacy
As a Software Engineer, I have utilised some AWS services and I wanted to solidify my experience working with AWS which is why I took the exam.

The exam format consists of:

  • 65 questions (multiple choice or multiple response)
  • Duration: 90 minutes
  • Format: Online or at a testing center

Study Resources

There are many resources available online for preparing for the exam. The ones I used and found most useful were:

1. AWS Skill Builder - AWS

AWS offers free and paid learning materials to study for the exam. They cover AWS Cloud concepts, core AWS services, Economics of the AWS Cloud, and Security and compliance in the AWS Cloud. I used the free version and took the exam prep official question set.

2. Ultimate AWS Certified Cloud Practitioner CLF-C02 - Stephane Maarek

This Udemy course is a must watch especially if you have no experience with using AWS services. It includes practical sessions of setting up and using AWS services which gets you familiar with them and in turn helps with taking the exam. It is impossible to pass any of the AWS exams without familiarity with AWS services. There is a practice exam at the end of the course to help you prepare for the actual exam. Practice Exams.

3. YouTube

There are different useful videos on YouTube with practice exam questions that are useful for getting familiar with exam style or question style. The videos aim to explain the concepts of the questions and not just provide answers to the questions. It is a good resource to prepare for the exam.

I took notes using Notion on the core AWS services and tips that were useful in answering questions on them. I highly recommend taking notes as you study so you can have everything in one place to go through on the exam day. The exam itself is pretty easy if you study really well.

If I had to pick just two resources, I would go with the Udemy course and the YouTube playlist. Both are more than enough to pass the exam. The good thing is both resources also have other AWS Exam sets and not just the Certified Cloud Practitioner if you are writing a different exam.

Exam Day Strategies

The AWS exam tests your ability to analyse scenarios and pick the best AWS solution. Here's what worked for me:

1. Be relaxed and settled: Skim through your notes on exam day to reinforce key concepts.

2. Time Management

  • First pass: Answer easy questions quickly
  • Flag difficult questions for review
  • Second pass: Revisit flagged questions

3. Elimination Technique: If unsure, eliminate obvious wrong answers and pick the best fit.

4. Look for AWS best practices: Prioritise scalability, security, and cost-effectiveness.

5. Be aware of tricky wording: AWS exams sometimes use similar-sounding terms to test understanding.

Final Thoughts & Advice

  • Start early and be consistent. AWS exams reward hands-on practice, not just theoretical knowledge.
  • Use multiple learning resources. Videos, whitepapers, hands-on labs, and practice tests together create the best learning experience.
  • Take notes! Having everything in one place to review on exam day can be invaluable.
  • Don't rush! The goal is to understand AWS concepts deeply, not just pass the exam.
