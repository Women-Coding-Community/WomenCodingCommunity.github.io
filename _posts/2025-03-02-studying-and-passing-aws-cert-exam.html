As a Software Engineer who has utilized some AWS services, I wanted to solidify my experience working with AWS by pursuing certification. In this blog post, I'll be sharing my experience preparing for and passing the AWS Certified Cloud Practitioner exam, along with resources and tips to help you succeed in your certification journey.

Exam Overview

The AWS Certified Cloud Practitioner Exam validates foundational, high-level understanding of AWS Cloud, services, and terminology. It's an excellent starting point on the AWS Certification journey for:

  • Individuals with no prior IT or cloud experience switching to a cloud career
  • IT professionals looking for foundational cloud literacy

The exam format consists of:

  • 65 questions (multiple choice or multiple response)
  • Duration: 90 minutes
  • Format: Online or at a testing center

Study Resources

There are many resources available online for preparing for the exam. The ones I used and found most useful were:

AWS Skill Builder

  • AWS offers free and paid learning materials to study for the exam
  • Covers AWS Cloud concepts, core AWS services, Economics of the AWS Cloud, and Security and compliance
  • I used the free version and took the official exam prep question set

Ultimate AWS Certified Cloud Practitioner CLF-C02 - Stephane Maarek

  • This Udemy course is a must-watch, especially if you have no experience with AWS services
  • Includes practical sessions for setting up and using AWS services
  • Offers practice exams to help prepare for the actual test

It's impossible to pass any AWS exam without familiarity with AWS services, and this course provides hands-on experience that's invaluable.

YouTube

  • Various useful videos with practice exam questions
  • Videos explain the concepts behind questions rather than just providing answers
  • Great resource for understanding exam and question styles

I took notes using Notion on the core AWS services and tips that were useful in answering questions about them. I highly recommend taking notes as you study so you can have everything in one place to review on exam day.

If I had to pick just two resources, I would choose the Udemy course and YouTube videos. Both are more than sufficient to pass the exam. An added benefit is that both resources also cover other AWS Exam sets beyond the Certified Cloud Practitioner.

Exam Day Strategies

The AWS exam tests your ability to analyze scenarios and pick the best AWS solution. Here's what worked for me:

1. Be relaxed and settled

  • Skim through your notes on exam day to reinforce key concepts
  • Arrive early if taking the exam at a testing center

2. Time Management

  • First pass: Answer easy questions quickly
  • Flag difficult questions for review
  • Second pass: Revisit flagged questions

3. Elimination Technique

  • If unsure, eliminate obviously wrong answers and pick the best fit
  • Look for keywords that point to specific AWS services

4. Look for AWS best practices

  • Prioritize scalability, security, and cost-effectiveness
  • Remember AWS's shared responsibility model

5. Be aware of tricky wording

  • AWS exams sometimes use similar-sounding terms to test understanding
  • Read questions carefully and identify what's really being asked

Final Thoughts & Advice

  • Start early and be consistent. AWS exams reward hands-on practice, not just theoretical knowledge.
  • Use multiple learning resources. Videos, whitepapers, hands-on labs, and practice tests together create the best learning experience.
  • Take notes! Having everything in one place to review on exam day can be invaluable.
  • Don't rush! The goal is to understand AWS concepts deeply, not just pass the exam.
